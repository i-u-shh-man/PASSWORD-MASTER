## **1\. PASSWORD MASTER D3F3NC3 (Defense Module)**


This module is designed as a **client-based tool** focused on **proactive defense**. Its primary function is to help users learn the current security standards and ensure their password measures security compliance before use.

### **Key Features:**

* Real-Time Feedback: Dynamic checklist SHOWS A TICK MARK WHEN THE REQUIREMENT OR CONDITION IS MET.  
* **Compliance:** Enforces minimum 10 characters, uppercase, digits, and symbols.  
* **High-Fidelity UI:** Dark theme, custom typography, and integrated visibility toggle for professional usability.

## **2\. PASSWORD MASTER OFF3NC3 (Offense Module)**


This module illustrates the methodology of **hackers and attackers** who acquire general information through social media and other mediums (OSINT). It focuses on **security auditing and testing**, via algorithmic dictionary generation to demonstrate **how easy it is to crack weak passwords**, and the resulting importance of making a strong one.

### **Key Features:**

* **Targeted Profiling:** Uses user-provided PII (Name, Pet, Year) as base words acquired via OSINT.  
* **Advanced Permutations:** Implements Leetspeak substitution (a \-\> 4, e \-\> 3, etc.) and intelligently appends common year suffixes (e.g., 2024, 24\) to create custom wordlists for brute-force attacks.  
* **Cracking Tool Ready:** Exports the unique, sorted wordlist in the standard .txt format required by tools like Hashcat.

* **Python 3.x**  
* **VS Code** (used for coding and testing)  
* Standard **tkinter** and **os** modules.
